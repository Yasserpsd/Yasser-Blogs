# Yasser Blogs

بلجن ووردبريس لعرض المقالات بشكل شيك ومميز في شبكة من 3 أعمدة، مع أزرار مشاركة سريعة على واتساب، لينكدإن، X (تويتر)، ونسخ الرابط.

## ✨ المميزات

- عرض المقالات في شبكة Grid شيكة ومتجاوبة (Responsive)
- أزرار مشاركة تظهر عند الـ Hover (واتساب، لينكدإن، X، نسخ الرابط)
- تصميم عصري بظلال ناعمة وحركات سلسة
- متجاوب بالكامل مع جميع الشاشات
- خفيف وسريع، بدون مكتبات خارجية

## 📦 التثبيت

1. حمّل البلجن كملف ZIP من GitHub.
2. ادخل على لوحة تحكم ووردبريس → **الإضافات → أضف جديد → رفع إضافة**.
3. ارفع الملف وفعّله.

## 🚀 الاستخدام

استخدم الشورت كود التالي في أي صفحة أو مقالة:

```
[yasser_blogs]
```

### خيارات الشورت كود

```
[yasser_blogs posts="6" columns="3" category="news" orderby="date" order="DESC"]
```

| الخيار | الوصف | القيمة الافتراضية |
|--------|-------|------------------|
| `posts` | عدد المقالات | 6 |
| `columns` | عدد الأعمدة (1، 2، 3، 4) | 3 |
| `category` | اسم التصنيف (slug) | فارغ (الكل) |
| `orderby` | الترتيب حسب | date |
| `order` | اتجاه الترتيب | DESC |
